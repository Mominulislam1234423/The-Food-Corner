# 🍔 The Food Corner - Restaurant Management & Food Delivery Platform

**The Food Corner** is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application designed for a seamless food ordering and restaurant management experience. Users can browse a rich menu, add/remove items dynamically from their cart, and track their selections via a dedicated Dashboard.

## ✨ Key Features

- **🔐 Firebase Authentication:** Secure user sign-in and sign-up using Email/Password and Social Login (Google Authentication).
- **🛒 Dynamic Cart System:** Users can add food items to their cart. Powered by `React Query (TanStack Query)`, the cart count in the Navbar updates in real-time without requiring a page refresh.
- **📊 Interactive User Dashboard:** Features a modern sidebar layout where authenticated users can view their selected items, calculate total pricing, and seamlessly manage or delete cart items.
- **🎨 Premium UI/UX:** Built using Tailwind CSS and DaisyUI, matching a clean, responsive layout with an elegant golden/tan (`#D1A054`) theme.
- **🔔 Real-time Notifications:** Utilizes `SweetAlert2` for beautiful pop-up confirmations when adding items, deleting items, or forcing login actions.

## 🛠️ Technologies Used

### Frontend
- **React.js** (Functional Components, Custom Hooks, Context API)
- **React Router** (v7 - Nested Routing with layout wrappers and `<Outlet />`)
- **@tanstack/react-query** (State Syncing, Caching & Data Fetching)
- **Tailwind CSS & DaisyUI** (Responsive Styling & Components)
- **Axios & Fetch API** (HTTP Request Handling)
- **React Icons & SweetAlert2**

### Backend & Database
- **Node.js** & **Express.js** (RESTful API Development)
- **MongoDB** (NoSQL Database Management)
- **Cors & Dotenv** (Security & Environment Variables Configuration)
